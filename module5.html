Module 5: Cracking WPA2 Passwords

🔓 Module 5: Breaking the Code – WPA2 Password Cracking

Welcome to the hacking part you’ve been waiting for — cracking the Wi-Fi password from the handshake you captured earlier. You’re now entering the brute force arena. 🥊

📁 What You Need

  • .cap file from Module 4 (your handshake)
  • A strong wordlist (like rockyou.txt)
  • Patience… and maybe a coffee ☕

🧠 What Is a Wordlist Attack?

You try a list of passwords one-by-one until one unlocks the Wi-Fi. Think of it like trying every key on a keyring. Except your CPU/GPU is doing the work for you while you pretend to be busy. 😌

⚙️ Cracking with aircrack-ng

The simplest tool to start with is aircrack-ng.

aircrack-ng capture.cap -w /usr/share/wordlists/rockyou.txt

If your password is in the list — boom 💥 you’ll see the key displayed like:

KEY FOUND! [ supersecret123 ]
🔐 Note: rockyou.txt is a great start, but for real-world routers, you might need better lists.

🦾 Want to Go Faster? Try Hashcat!

If you’ve got a decent GPU, hashcat is like aircrack-ng on steroids.

Step 1: Convert .cap to .hccapx

cap2hccapx capture.cap capture.hccapx

Step 2: Run Hashcat

hashcat -m 2500 capture.hccapx rockyou.txt --force

You’ll get real-time stats, speed, and hopefully a password at the end. Just try not to fry your GPU. 🔥

🧪 Real-World Fun

You cracked Bob’s COFFEE_FREEWIFI network from earlier. Turns out his password was latte4life. ☕💻 Lesson? People use dumb passwords. Hackers thrive on that laziness.

📦 Summary

  • Use aircrack-ng for easy cracking with wordlists
  • Hashcat is faster and GPU-based for big lists
  • Rockyou.txt is good, but custom or regional lists are better
  • Passwords like 12345678 still exist... thanks humans
